# rota_da_beleza
Aplicativo desenvolvido para fins acadêmicos e projetos comerciais.

Método para efetuar debug

acessar a pasta do expo
No terminal execute os comandos abaixo

1 ->  cd expo_app\rota_da_beleza   //entrar na pasta rota_da_beleza
2 ->  npx expo start //executar o expo localmente, isso vai gerar um QR Code para acesso

OBS: o qr code deve ser lido através do aplicativo Expo GO disponível na playStore
